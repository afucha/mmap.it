## Releasing
```bash
yarn dist
```

## Contributing
### Running the development server
To run the developement server you will need node 13 and yarn installed.

#### Clone Repo

```bash
git clone https://github.com/pdepip/mmap.it
```

#### Install dependencies

```bash
yarn 
```

#### Run the server

```bash
yarn start-dev
```
